Игра "В погоне за стипендией"

  • Введите свое имя и нажмите кнопку начать
  • Уничтожайте монстров по ходу прохождения игры
  • Исользуйте "правую стрелку" и "левую стрелку" для перемещения
  • Используй кнопку "пробел" для того, чтобы ударить противника
  • Для использования умений нажимайте 1, 2, 3 и 4. Навыки имеют время перезарядки!
